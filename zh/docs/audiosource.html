AudioSource

AudioSource

附加到实体的空间音频播放组件。控制循环、音量、音调和播放状态。

#include <GryceRC/GryceAudioSource.h>

struct AudioSource {
    AudioClipHandle clip;
    bool loop = false;
    bool auto_play = false;
    float volume = 1.0f;
    float pitch = 1.0f;
    float min_distance = 1.0f;  // 3D 衰减起始距离
    float max_distance = 100.0f; // 3D 衰减结束距离

    void play();
    void pause();
    void stop();
    bool is_playing() const;

    void set_time(float seconds);
    float time() const;
    float duration() const;
};

示例

auto entity = world.create_entity("Music");
auto& source = entity.add_component<AudioSource>();
source.clip = audio.load_clip("res:/audio/bg.mp3");
source.loop = true;
source.volume = 0.5f;
source.play();

设计原理

AudioSource 是 gryce-engine 中的音频源组件，附加到 Entity 上用于播放声音。它支持 3D 空间定位：AudioEngine 在更新时读取 AudioSource 所在 Entity 的 Transform 位置，计算距离衰减和方向声像。AudioSource 支持循环播放、音量控制、播放/暂停/停止，以及播放进度查询。每个 AudioSource 对应 miniaudio 中的一个声音源（voice），多个 AudioSource 可以同时播放。

空间音频

AudioSource 的 min_distance 和 max_distance 控制 3D 衰减范围。在 min_distance 内音量最大，超过 max_distance 后音量衰减为 0。miniaudio 的 attenuation model 使用 inverse distance，适合大多数游戏场景。对于非空间音频（如背景音乐），将 min_distance 设为极大值即可禁用衰减。

注意事项

  • AudioSource 的 clip 必须在 AudioEngine 中加载，否则 play() 会失败。
  • loop 在音频到达末尾时自动重新开始，不需要手动调用 play()。
  • set_time 以秒为单位，设置值超出 duration 时会被截断。
  • 多个 AudioSource 可以同时播放同一个 clip，不共享底层 voice。
