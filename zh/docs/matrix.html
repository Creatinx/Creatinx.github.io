Matrix

Matrix4f

列主序 4x4 矩阵。存储布局：m[col * 4 + row]。提供完整的变换工厂方法。

#include <GryceRC/GryceMath.h>

struct Matrix4f {
    float m[16];

    static Matrix4f identity();
    static Matrix4f zero();

    static Matrix4f translate(float x, float y, float z);
    static Matrix4f translate(const Vector3f& v);
    static Matrix4f scale(float x, float y, float z);
    static Matrix4f scale(const Vector3f& v);
    static Matrix4f rotate_x(float degrees);
    static Matrix4f rotate_y(float degrees);
    static Matrix4f rotate_z(float degrees);
    static Matrix4f rotate(const Vector3f& axis, float degrees);

    static Matrix4f ortho(float left, float right, float bottom, float top, float near, float far);
    static Matrix4f perspective(float fov_degrees, float aspect, float near, float far);
    static Matrix4f look_at(const Vector3f& eye, const Vector3f& center, const Vector3f& up);

    Matrix4f operator*(const Matrix4f& other) const;
    Vector4f operator*(const Vector4f& v) const;
    Vector3f transform_point(const Vector3f& p) const;
    Vector3f transform_vector(const Vector3f& v) const;

    Matrix4f transposed() const;
    Matrix4f inverse() const;
    float determinant() const;

    float* data();
    const float* data() const;
    float operator()(int row, int col) const;
    float& operator()(int row, int col);
};

示例

Matrix4f model = Matrix4f::translate({2,0,0}) * Matrix4f::rotate_y(45.0f);
Matrix4f view = Matrix4f::look_at({0,2,5}, {0,0,0}, {0,1,0});
Matrix4f proj = Matrix4f::perspective(60.0f, 16.0f/9.0f, 0.1f, 100.0f);
Matrix4f mvp = proj * view * model;

设计原理

Matrix4f 是 gryce-engine 中的列主序 4x4 矩阵，存储为 float m[16]。列主序意味着 m[col*4 + row] 访问元素 (row, col)。这种布局与 OpenGL 的 glUniformMatrix4fv 和 Vulkan 的 push constant 兼容，可以直接传递而不需要转置。Matrix4f 提供所有常见的变换工厂方法：translate、scale、rotate、ortho、perspective、look_at。所有角度参数使用度数而非弧度。

矩阵运算

Matrix4f 的乘法使用标准矩阵乘法，时间复杂度 O(n^3) 其中 n=4。inverse() 使用伴随矩阵法，对于标准变换矩阵（TRS）足够高效。对于一般矩阵，inverse() 的性能和精度可能不如专门的线性代数库（如 Eigen）。determinant() 返回矩阵的行列式，用于判断矩阵是否可逆。transposed() 返回转置矩阵，用于法线矩阵计算。

注意事项

  • Matrix4f 使用列主序，与 DirectX 的行主序不同，与 Shader 中的 mat4 一致。
  • perspective 的 fov 参数是垂直视角（度数），aspect 是宽高比。
  • look_at 的 up 向量不需要归一化，内部会自动处理。
  • inverse() 在矩阵不可逆时（行列式接近 0）返回零矩阵，不抛出异常。
