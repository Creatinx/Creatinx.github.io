BuiltInPhysicsWorld2D

零依赖的轻量级 2D 物理。AABB 宽相 + SAT 窄相。仅支持圆形和 AABB 形状。显式 opt-in，不需要外部库。

#include <GryceRC/GryceBuiltInPhysics.h>

class BuiltInPhysicsWorld2D : public IPhysicsWorld2D {
public:
    bool init() override;
    void shutdown() override;

    void step(float dt) override;
    void set_gravity(const Vector2f& gravity) override;

    BodyHandle create_body(const BodyDesc2D& desc) override;
    void destroy_body(BodyHandle handle) override;

    void set_position(BodyHandle handle, const Vector2f& pos) override;
    void set_velocity(BodyHandle handle, const Vector2f& vel) override;

    Vector2f get_position(BodyHandle handle) const override;
    Vector2f get_velocity(BodyHandle handle) const override;

    void add_force(BodyHandle handle, const Vector2f& force) override;
    void add_impulse(BodyHandle handle, const Vector2f& impulse) override;

    RaycastResult2D raycast(const Vector2f& from, const Vector2f& to) const override;
    std::vector<CollisionResult2D> check_collision(BodyHandle handle) const override;

    void set_sleep_threshold(float velocity_sq); // 默认 0.01
    void set_iterations(int position_iters, int velocity_iters); // 默认 3, 8
};

// 支持的形状
struct CircleShape { float radius; };
struct AABBShape { Vector2f half_extents; };
struct PolygonShape { std::vector<Vector2f> vertices; };

设计原理

BuiltInPhysicsWorld2D 是 gryce-engine 的零依赖 2D 物理后端，用于不需要 Box2D 的轻量级场景。它使用 AABB 宽相（broadphase）检测潜在碰撞对，然后使用 SAT（Separating Axis Theorem）窄相（narrowphase）进行精确碰撞检测。支持 Circle 和 AABB 形状，Polygon 形状使用凸包简化。BuiltInPhysicsWorld2D 的 solver 使用简单的 impulse-based 方法，适合少量刚体（< 100）的场景。

性能特性

BuiltInPhysicsWorld2D 的 AABB 宽相使用简单的空间分割（每帧重新排序），时间复杂度为 O(n log n)，其中 n 是刚体数量。对于 n < 100 的场景，性能与 Box2D 相当。但对于大量刚体或复杂关节，Box2D 的 island-based solver 更高效。BuiltInPhysicsWorld2D 不支持关节，所有连接必须通过代码手动模拟。

注意事项

  • BuiltInPhysicsWorld2D 仅支持 Circle、AABB 和凸 Polygon 形状。
  • sleep_threshold 控制刚体何时进入休眠，休眠刚体不参与求解，节省 CPU。
  • iterations 控制 solver 的迭代次数，值越高越稳定但越慢。
  • BuiltInPhysicsWorld2D 不支持 CCD（连续碰撞检测），快速移动的物体可能穿透。
