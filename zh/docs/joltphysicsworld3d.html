JoltPhysicsWorld3D

JoltPhysicsWorld3D

使用 Jolt Physics 5.2 的 3D 刚体物理。支持静态、动态和运动体。包含射线检测和碰撞检测。

#include <GryceRC/GryceJoltPhysics.h>

class JoltPhysicsWorld3D : public IPhysicsWorld3D {
public:
    bool init() override;
    void shutdown() override;

    void step(float dt) override;
    void set_gravity(const Vector3f& gravity) override;

    BodyHandle create_body(const BodyDesc& desc) override;
    void destroy_body(BodyHandle handle) override;

    void set_position(BodyHandle handle, const Vector3f& pos) override;
    void set_rotation(BodyHandle handle, const Quaternion& rot) override;
    void set_velocity(BodyHandle handle, const Vector3f& vel) override;

    Vector3f get_position(BodyHandle handle) const override;
    Vector3f get_velocity(BodyHandle handle) const override;

    void add_force(BodyHandle handle, const Vector3f& force) override;
    void add_impulse(BodyHandle handle, const Vector3f& impulse) override;

    RaycastResult raycast(const Vector3f& from, const Vector3f& to) const override;
    std::vector<CollisionResult> check_collision(BodyHandle handle) const override;
};

struct BodyDesc {
    BodyType type = BodyType::Dynamic; // Static, Dynamic, Kinematic
    ShapeHandle shape;
    Vector3f position;
    float mass = 1.0f;
    PhysicalMaterial material;
};

设计原理

JoltPhysicsWorld3D 是 gryce-engine 的 3D 物理后端，基于 Jolt Physics 5.2。它实现了 IPhysicsWorld3D 接口，提供刚体动力学、碰撞检测、射线检测和约束系统。JoltPhysicsWorld3D 在 init() 中创建 Jolt 的 PhysicsSystem、BodyInterface 和 ContactListener。在 step() 中，它调用 PhysicsSystem::Update 以固定时间步长推进物理模拟。Jolt 的 job system 使用 std::thread 实现多线程物理模拟，默认使用所有可用 CPU 核心。

刚体类型

Jolt 支持三种刚体类型：Static（不动，质量无穷大）、Dynamic（受力和碰撞影响）、Kinematic（由代码控制位置，不受力影响但影响其他刚体）。在 BodyDesc 中通过 type 字段指定。PhysicalMaterial 的 friction 和 restitution 在 Jolt 中映射到 Body 的 friction 和 restitution_scale。

注意事项

  • Jolt 的 step() 必须在固定时间步长下调用，否则模拟不稳定。
  • raycast 返回的 RaycastResult 包含 hit body、hit point 和 hit normal。
  • add_force 在世界坐标系中施加力，add_impulse 施加冲量（瞬时力）。
  • shape 在 Jolt 中通过 ShapeHandle 引用，由 JoltPhysicsWorld3D 内部管理。
