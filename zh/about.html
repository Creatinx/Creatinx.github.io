关于 Gryce Engine

C++23 · CMake · OpenGL 4.6 / Vulkan 1.2

Gryce Engine 是一个从零手写的 C++ 游戏引擎，主要用于学习图形 API、编译器周边工具与引擎架构。没有使用现成引擎框架。

目前已实现 OpenGL 4.6 和早期 Vulkan 1.2 后端（简单 RHI 抽象）、自定义 ECS（稀疏集存储）、可插拔物理/音频模块。大量功能仍在实验阶段。

开发环境：Windows + MSYS2 UCRT64 GCC (C++23) + CMake + Ninja。尚未达到生产可用状态。
