GLShader

GLShader

Wraps vertex and fragment shader compilation, uniform uploads, and program linking. Reports compilation errors via spdlog.

#include <GryceRC/GryceGLShader.h>

class GLShader {
public:
    bool compile(const std::string& vertex_src, const std::string& fragment_src);
    void bind();
    void unbind();

    void set_int(const std::string& name, int value);
    void set_float(const std::string& name, float value);
    void set_vec2(const std::string& name, float x, float y);
    void set_vec3(const std::string& name, float x, float y, float z);
    void set_vec4(const std::string& name, float x, float y, float z, float w);
    void set_mat4(const std::string& name, const float* m);
    void set_mat4(const std::string& name, const Matrix4f& m);

    int get_uniform_location(const std::string& name); // returns -1 if missing
    uint32_t program_id() const;
};

Example

GLShader shader;
shader.compile(R"(
    #version 460 core
    layout(location=0) in vec3 a_pos;
    uniform mat4 u_mvp;
    void main() { gl_Position = u_mvp * vec4(a_pos, 1.0); }
)", R"(
    #version 460 core
    out vec4 frag_color;
    void main() { frag_color = vec4(1.0); }
)");
shader.bind();
shader.set_mat4("u_mvp", mvp.data());

设计原理

GLShader 封装了 OpenGL shader program 的完整生命周期：顶点/片段着色器编译、链接、uniform 查找和上传。它在 compile() 中创建 shader object、编译源码、链接 program，并将错误信息通过 spdlog 报告。uniform 位置在首次 set_* 调用时缓存，后续调用直接复用，避免重复的 glGetUniformLocation 查询开销。

Uniform 缓存

GLShader 内部维护一个 unordered_map<std::string, int> 来缓存 uniform 位置。当调用 set_int / set_float / set_mat4 时，先查找缓存，如果未命中则调用 glGetUniformLocation 并插入缓存。这种设计将 uniform 设置的开销从 O(n) 降低到 O(1)（均摊），其中 n 是 program 中的 uniform 数量。

注意事项

  • compile() 必须在 GLBackend 初始化之后调用，因为 shader program 需要活跃的 OpenGL 上下文。
  • get_uniform_location() 在 uniform 不存在时返回 -1，调用者应该检查返回值。
  • set_mat4 要求传入的 float* 指向 16 个连续 float（列主序）。
  • GLShader 析构时自动调用 glDeleteProgram。

A very early-stage C++23 game framework. No editor. Not built from scratch — depends on GLFW, GLEW, Vulkan SDK, miniaudio, Jolt, Box2D, Assimp, and others. Handwritten parts: ECS, RHI, math, scene.
